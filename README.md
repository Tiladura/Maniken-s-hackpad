# 6-KEY HACKPAD WITH ENGRAVINGS OF MY CATS NAME
## 2x3 hackpad which can be useful for creating macros or shortcuts for personal use. Convinient key placement, cute cat ascii images and my cats name are engraved(cats name - Tila).
## Features:
- 6 keys 2x3 placement
- xiao rp2040 microcontroller
